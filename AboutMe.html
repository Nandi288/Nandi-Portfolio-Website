I am a 22-year-old final-year student completing my Diploma in ICT: Application Development at the Durban University of Technology. My focus is building full-stack solutions — from database architecture and C# back-end logic to clean, functional front-end interfaces.

My core stack is C#, ASP.NET MVC, and Microsoft SQL Server. I have hands-on experience designing relational databases, writing stored procedures, and wiring them up to MVC applications deployed on Azure App Services.

Right now I'm expanding into ASP.NET Core — modernising the way I build APIs and web applications — and I'm teaching myself Figma to strengthen the UI/UX side of my projects. I believe that a full-stack developer who can prototype in Figma before writing a line of code builds better products, faster.

Goal: To pursue an Advanced Diploma in ICT: Application Development and grow into a full-stack developer who specialises in .NET ecosystems (ASP.NET Core, Entity Framework), cloud deployment on Azure, and design-informed front-end development — bridging the gap between engineering and user experience.

I enjoy collaborative projects where I can contribute to both the technical architecture and the design thinking. If you're looking for a motivated junior developer who ships working software and keeps learning — let's connect.
