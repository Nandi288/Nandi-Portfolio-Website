Student Management System

Collaborated in a group to build a full-stack Student Management System on ASP.NET Core MVC (.NET 8), deployed to Microsoft Azure. Features OAuth 2.0 authentication via GitHub and Google, Role-Based Access Control (RBAC), and full CRUD operations for student records. Profile pictures are stored securely in Azure Blob Storage with time-limited SAS tokens, and student data is persisted in Azure Cosmos DB.
